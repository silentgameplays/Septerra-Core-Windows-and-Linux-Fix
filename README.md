# Septerra-Core-Windows-and-Linux-Fix
Septerra Core Windows and Linux Fix

Windows:
1. Install Septerra Core from Steam/GoG
2. Download all files in this repo
3. Drop the files into your C:\Program Files (x86)\Steam\steamapps\common\Septerra Core folder, overwriting if necessary
4. You can make adjustments to the Septera.cfg file for scaling,resolution and output the current settings are 640x480 stretched to fill fullscreen

Linux

1. Install Septerra Core from Steam/GoG
2. Set compatibility to Proton Version 9.04 or download GE PRoton versions and check which one of the works better GE 10.20 or 10.25 should work,native Linux version is wonky
3. Download all files in this repo
4. Drop the files into your /home/username/.local/share/Steam/steamapps/common/Septerra Core folder, overwriting if necessary
5. You can make adjustments to the Septera.cfg file for scaling,resolution and output the current settings are 640x480 stretched to fill fullscreen
